Umiejętności

Tworzę strony zgodne z semantyką standardu HTML 5 oraz dbam o czystość kodu. Dzięki stylom i animacjom stworzonym w CSS strona sprawia, że chcesz pozostać na niej dłużej. Tworzę responsywne layouty dzięki CSS grid, a stosowana przeze mnie metodologia BEM poprawia nie tylko czytelność kodu CSS, ale także HTML. W swoich projektach korzystam również z JavaScript oraz JQuery, jednak znajomość ich jest na nieco niższym poziomie niż wcześniej wymienione technologie. W wolnym czasie wciąż dąże do opanowania ich.

Wiedzę czerpałem m. in. z tutoriali pasja-informatyki oraz ksiązek: Duckett J. - HTML i CSS. Zaprojektuj i zbuduj witrynę WWW ; Freeman E. - Programowanie w JavaScript. Rusz głową!

W tworzeniu responsywnych projektów pomaga mi framework Bootstrap. Moją pracę przyspiesza również preprocesor SASS, dzięki czemu struktura dużych projektów jest czytelna. Korzystam także z systemu kontroli wersji GIT.

Umiejętności zdobyłem dzięki kursom Udemy - Advanced CSS and Sass: Flexbox, Grid, Animations and More., dokumentacji oraz innym ogólno dostępnym tutorialom.

W utrzymaniu porządku pomaga mi również narzędzie jakim jest webpack oraz menadżer pakietów NPM.
